# Student Performance Prediction using Machine Learning

## Overview
This project focuses on predicting student academic performance based on factors such as weekly self-study hours, attendance percentage, and class participation using machine learning techniques.

## Problem Statement
To build a regression-based machine learning model that can predict a student's total score using academic and behavioral factors.

## Dataset
The dataset contains numerical features representing student study habits, attendance, and engagement.

## Methodology
- Data loading and exploratory data analysis
- Feature selection and preprocessing
- Train-test split
- Model training using Linear Regression
- Model evaluation using RMSE and R² score
- Interpretation of results

## Tools & Technologies
- Python
- Pandas
- NumPy
- scikit-learn
- Google Colab

## Results
The model achieved a satisfactory R² score, indicating that study habits and attendance have a significant impact on student performance.

## Conclusion
This project demonstrates an end-to-end machine learning workflow, from data analysis to model evaluation and interpretation.
